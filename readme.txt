# クリックRPG（仮）

まだ仮題ですが、主にマウスのクリック操作で遊ぶRPG（ロールプレイングゲーム）です。

## 操作説明

### 戦闘シーン以外の操作
戦闘シーン以外では基本的にマウスで各ボタンをクリックするだけです。
