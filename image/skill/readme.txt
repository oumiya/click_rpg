特技一覧
・HP回復:00_heal.png
　→ 薬草の節約になる。クールタイム長め。
・防御の判定が甘くなるスキル:01_guard_range_up.png
　→ GOOD が PERFECT に、POOR が GOOD になる
・2倍攻撃:02_super_attack.png
　→ 武器の属性付での攻撃力2倍攻撃
・攻撃力UP:03_up_attack.png
　→ 一定時間攻撃力がUPする。150%上昇とする。持続時間よりクールタイムの方が長め
・防御力UP:04_up_guard.png
・敵の攻撃力DOWN:05_down_attack.png
・敵の防御力DOWN06_down_guard.png
　→ 同上
・スタン:07_shout.png
　→ 敵の攻撃を封じる。攻撃マークが既に出ていても攻撃マークを消す。連続攻撃を持つ敵に有効。
・属性反転:08_reverse.png
　→ 自分の持つ武器の属性を反転させる。氷なら火に、闇なら光に。クールタイムなし。
